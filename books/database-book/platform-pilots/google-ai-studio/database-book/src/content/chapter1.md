# Chapter 1: Introduction to React

React is a declarative, efficient, and flexible JavaScript library for building user interfaces. It lets you compose complex UIs from small and isolated pieces of code called "components".

## Components

Components are the building blocks of React. They are like JavaScript functions that return HTML elements.

```tsx
function Welcome(props) {
  return <h1>Hello, {props.name}</h1>;
}
```

## Props

Props are short for properties. They are used to pass data from one component to another.

## State

State is a way to store data that can change over time. When state changes, React re-renders the component.
