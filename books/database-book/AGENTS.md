# Database Book Repo Instructions

The repository-side edit tracking files are canonical for the database book.

## Edit tracking

- `books/database-book/.edits/edit-log.md` is the rolling book-wide edit log.
- `books/database-book/.edits/chapter-tracker.md` is the chapter and book task tracker.
- `books/database-book/.edits/chNN-edits.md` stores per-chapter edit notes.

## Workflow

- Update these repo-side files automatically during normal book-edit work.
- Do not create new Google Drive `.docs/.edits` files for new work.
- Treat Drive-side edit logs as historical only.
